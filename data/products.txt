Product Documentation
=====================

Acme Corp Products Overview
---------------------------

### 1. CodeFlow (Flagship Product)

CodeFlow is an AI-powered code review and analysis platform that integrates directly into
your GitHub, GitLab, or Bitbucket workflow.

Key Features:
- Automated code review with AI suggestions
- Security vulnerability detection
- Performance bottleneck identification
- Custom rule enforcement
- Integration with Slack, Jira, and Linear

Pricing:
- Free tier: Up to 5 developers, 500 reviews/month
- Pro: $29/developer/month - unlimited reviews, priority support
- Enterprise: Custom pricing - SSO, audit logs, dedicated support

Supported Languages:
TypeScript, Python, Go, Rust, Java, Ruby, PHP, Swift, Kotlin, C++

### 2. DeployBot

DeployBot automates deployment pipelines with AI-assisted rollback detection and
intelligent canary deployments.

Key Features:
- One-click deployments to AWS, GCP, Azure, and Vercel
- Automatic rollback on anomaly detection
- Canary deployment with traffic shifting
- Deployment analytics and DORA metrics tracking

Pricing:
- Free tier: 100 deployments/month
- Pro: $49/month - unlimited deployments, advanced analytics
- Enterprise: Custom pricing - multi-region, SLA guarantees

### 3. DocGen

DocGen generates and maintains API documentation from your codebase automatically.

Key Features:
- OpenAPI/Swagger generation from code annotations
- Auto-updating documentation on code changes
- Interactive API playground
- Multi-version documentation support

Pricing:
- Free tier: 1 project, 1000 API endpoints
- Pro: $19/month - unlimited projects
- Enterprise: Custom pricing - custom branding, SSO

### Getting Started

To get started with any Acme product:

1. Sign up at https://app.acmecorp.com
2. Connect your version control provider (GitHub/GitLab/Bitbucket)
3. Install the appropriate integration
4. Configure your first project using the setup wizard
5. Invite your team members

API Access:
All products expose REST APIs. Generate API keys from your dashboard at
https://app.acmecorp.com/settings/api-keys

Rate Limits:
- Free tier: 100 requests/minute
- Pro tier: 1000 requests/minute
- Enterprise: 10,000 requests/minute
