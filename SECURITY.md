# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not** open a public
issue. Instead, report it privately by emailing
**casper-justus@users.noreply.github.com**.

You should receive a response within 72 hours. If you don't, please follow up
to ensure the message was received.

We'll keep you informed as the issue is investigated and resolved.
