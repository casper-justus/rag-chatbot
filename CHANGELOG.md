## [1.0.0] - 2026-05-01
### Added
- Context-aware RAG chatbot using LangChain, Gemini API, and Qdrant vector database
- Document ingestion pipeline with chunking and embedding generation
- Qdrant vector store integration for semantic similarity search
- Gemini API integration for context-aware response generation
- LangChain retrieval chain connecting vector search with LLM responses
- Environment-based configuration for API keys and Qdrant connection
